Class VantInterface

Interface to any vantage weather station (Vue, Pro, Pro 2). Provides useful methods to access realtime weather data from your weather station's console. The device must be connected serially. To interact with the weather station create an instance of this class using VantInterface.create.

This interface is limited to station independent features. Use VantPro2Interface, VantProInterface and VantVueInterface for station dependent features.

Static create

  • create(settings: MinimumVantInterfaceSettings): Promise<VantInterface>
    • Defined in src/interfaces/VantInterface.ts:206

    Creates an interface to your vantage console (Vue, Pro, Pro 2) using the passed settings. The device should be connected serially. The passed path specifies the path to communicate with the console. On Windows paths like COM1, COM2, ... are common, on osx/linux devices common paths are /dev/tty0, /dev/tty2, ...

    Weather station dependent functionality (e.g. firmware version code for Vantage Pro 2 / Vue) is not supported on this interface. Use VantPro2Interface, VantProInterface and VantVueInterface for station dependent features.

    example
    const device = await VantInterface.create({ path: "COM3" });

    const highsAndLows = await device.getHighsAndLows();
    inspect(highsAndLows);

    await device.close();
