Class VantVueInterface

Interface to the Vantage Vue weather station. Is built on top of the VantPro2Interface.

Offers station dependent features like getRichRealtimeData, {@link getLOOP}, getLOOP2 and getFirmwareVersion.

Static create

  • create(settings: MinimumVantInterfaceSettings): Promise<VantVueInterface>

  • Overrides VantPro2Interface.create

    • Defined in src/interfaces/VantVueInterface.ts:28

    Creates an interface to your vantage vue weather station using the passed settings. The device should be connected serially. The passed path specifies the path to communicate with the weather station. On Windows paths like COM1, COM2, ... are common, on osx/linux devices common paths are /dev/tty0, /dev/tty2, ...

    example
    const device = await VantVueInterface.create({ path: "COM3" });

    await device.open();
    await device.wakeUp();

    const highsAndLows = await device.getHighsAndLows();
    inspect(highsAndLows);

    Parameters

    • settings: MinimumVantInterfaceSettings

      the settings

    Returns Promise<VantVueInterface>
